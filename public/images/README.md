# Huong dan hinh anh

Dat hinh anh san pham theo cau truc:

```text
public/images/products/<slug>/cover.jpg
public/images/products/<slug>/detail-1.jpg
public/images/products/<slug>/detail-2.jpg
```

`<slug>` phai trung voi slug trong `data/seed.json`.

Duoi file ho tro:

- `webp`
- `jpg`
- `jpeg`
- `png`
